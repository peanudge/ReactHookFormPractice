# React Hook Form Kata Project

Purpose: pratice react-hook-form library 
 

- Install MUI (Material UI)

```bash
npm install @mui/material @emotion/react @emotion/styled
```

- Install ReactHookForm

```bash
npm install react-hook-form
```
